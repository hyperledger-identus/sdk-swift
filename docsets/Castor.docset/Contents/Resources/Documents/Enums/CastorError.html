CastorError

public enum CastorError : KnownPrismError
extension CastorError: Equatable

An enum representing a known error in a Prism API response.

Note

When an error occurs during an API request/response cycle, the server may return an error object in the response. This object may include an error code and an error message. If the error received conforms to the KnownPrismError protocol, it will be classified as a known error.
