Apollo

public protocol Apollo

The Apollo protocol defines the set of cryptographic operations that are used in the Atala PRISM architecture.

  • createRandomMnemonics()

    createRandomMnemonics creates a random set of mnemonic phrases that can be used as a seed for generating a private key.

    Declaration

    Swift

    func createRandomMnemonics() -> [String]

    Return Value

    An array of mnemonic phrases

  • createSeed(mnemonics:passphrase:)

    createSeed takes in a set of mnemonics and a passphrase, and returns a seed object used to generate a private key. This function may throw an error if the mnemonics or passphrase are invalid.

    Throws

    An error if the mnemonics or passphrase are invalid

    Declaration

    Swift

    func createSeed(mnemonics: [String], passphrase: String) throws -> Seed

    Parameters

    mnemonics

    An array of mnemonic phrases
    passphrase

    A passphrase used to enhance the security of the seed

    Return Value

    A seed object

  • createRandomSeed()

    createRandomSeed creates a random seed and a corresponding set of mnemonic phrases.

    Declaration

    Swift

    func createRandomSeed() -> (mnemonic: [String], seed: Seed)

    Return Value

    A tuple containing an array of mnemonic phrases and a seed object

  • createPrivateKey(parameters:)

    Creates a private key using the provided parameters.

    Throws

    An error if the private key could not be created. The specific error will depend on the underlying key creation process.

    Declaration

    Swift

    func createPrivateKey(parameters: [String : String]) throws -> PrivateKey

    Parameters

    parameters

    A dictionary where keys are KeyProperties values (expressed as Strings) and their respective values are associated string representations. This can include properties such as kid (key ID), type (key type), algorithm (algorithm used), curve (curve used), seed (seed value), rawKey (raw key value), and others depending on the specific type of private key being created.

    Return Value

    A PrivateKey constructed from the parameters.
