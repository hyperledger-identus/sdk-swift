  • UnknownError

    An enum representing an unknown error in a Prism API response.

    Note

    When an error occurs during an API request/response cycle, the server may return an error object in the response. This object may include an error code, an error message, and possibly an array of underlying errors. If the error received does not conform to the KnownPrismError protocol, it will be classified as an UnknownPrismError.

    See also
